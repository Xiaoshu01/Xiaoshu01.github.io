About Index Page

网站首页，通过从网页布局，top,left content,centre，right content，footer分别介绍。

Top

Resources ： 即left content和footer各种资源分类，为方便从其他网页能快速跳转下一个资源。

How To ： 关于某个资源类的一些使用技巧。

Entrez ： 搜索，可在前面限制数据库，在搜索的时候可以通过限制，例如查找某个人近来发表的所有文献，可在PubMed数据库，搜索(具体工作单位[Affiliation]) AND 发表作者[Author]进行限制搜索，通过AND,OR,NOT进行逻辑组合，还可以通过搜索框下Advanced进行高级搜索。

Left Content

左侧主要是各资源的分类，通过第一条A-Z可查看所有没有分类列表

Centre

中间包括一些数据获取，提交，API开发，学习教程和分析工具等，在Analyze主要为实用的工具，例如Blast，Taxonomy Browser等。

Right Content

首先是NCBI最常访问的板块，包括

PubMed文献数据库；

Bookshelf可得到一些英文教材原版或者NCBI使用手册；

PubMed Central比PubMed可获取论文全文信息，简称PMC，而PubMed一般只有摘要等信息，而且相同搜索关键字在PubMed Central数据库得到结果数目大于PubMed，原因是因为匹配关键字全文内容比摘要概率更大；

BLAST，局部序列比对工具（一般输入query有多条的时候一起输入，而不是分开输入，由于NCBI对于提交者不管数据多大都一视同仁；比对选择数据库时候wgs,sra数据比较全，但注释差）；以及其他常用核酸蛋白数据库等

Footer

底部主要重复了前面的资源，其中Featured模块包括了一些对大众不常用，但在某些领域常用的快速入口。

MyNCBI

注册NCBI账号，可保存个人在NCBI上搜索以及访问记录，在搜索框下的create alert和rss可以追踪某个领域的更新信息。

E-Utilities

实现自动化的大批量的从Entrez数据库下载数据,并且提供了几种常用的语言以供选择，像Perl, Python, Java, 和 C++等.

利用win10自带Hyper-V安装Ubuntu以及安装使用本地blast

Ubuntu下载地址

利用Hyper-V安装Ubuntu(source:CSDN)

家庭版Win10安装Hyper-V虚拟机(source:Windows系统粉)

1.将如下代码添加到记事本中，并另存为Hyper-V.cmd文件。

pushd "%~dp0"

dir /b %SystemRoot%\servicing\Packages\*Hyper-V*.mum >hyper-v.txt

for /f %%i in ('findstr /i . hyper-v.txt 2^>nul') do dism /online /norestart /add-package:"%SystemRoot%\servicing\Packages\%%i"

del hyper-v.txt

Dism /online /enable-feature /featurename:Microsoft-Hyper-V-All /LimitAccess /ALL

2.桌面上，我们找到并右键点击【Hyper-V.cmd】文件图标，在右键菜单中点击：以管理员身份运行。

3.然后弹出一个 用户帐户控制 - Windows命令处理程序 对话框，点击：是。

4.然后系统自动进行Windows命令处理，我们等待处理完成以后，在最末处输入：Y，电脑自动重启，进行配置更新。

5.电脑重启之后，在window管理中可以看到已经成功安装了Hyper-V。

Quick Start

Install Blast Program

点击系统搜索按钮，输入terminal打开命令终端terminal(快捷键：ctrl+alt+t)

切换到下载目录

下载blast可执行程序

解压缩tar.gz文件

移动文件到我安装本地程序的目录

$ mv ncbi-blast-2.7.1+ ~/local/app/ 

$ cd ~/local/app/  

$ mv ncbi-blast-2.7.1+ blast

将BLAST+可执行程序所在目录（bin）的绝对路径加入到环境变量$PATH，通过pwd命令获取blast所在目录路径

$ echo "export PATH=blast所在路径/bin:\$PATH" >> ~/.bashrc

$ source ~/.bashrc #使配置生效 

$ blastn -version #查看配置是否成功

配置本地Blast库

（source：CSDN）

根据自己习惯创建存放核酸蛋白质库数据的文件目录

在根目录下创建data/blast

在blast文件夹下创建并且编辑.ncbirc全局配置文件，内容如下：

Start the section for BLAST configuration

[BLAST]

; Specifies the path where BLAST databases are installed
BLASTDB=/home/用户名/data/blast

; Specifies the data sources to use for automatic resolution

; for sequence identifiers
DATA_LOADERS=blastdb

; Specifies the BLAST database to use resolve protein sequences
BLASTDB_PROT_DATA_LOADER=/home/用户名/data/blast/nr

; Specifies the BLAST database to use resolve protein sequences
BLASTDB_NUCL_DATA_LOADER=/home/用户名/data/blast/nt

BATCH_SIZE=10G

; Windowmasker settings

[WINDOW_MASKER]
WINDOW_MASKER_PATH=home/用户名/data/blast/windowmasker

; end of file
